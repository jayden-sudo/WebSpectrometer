Web Spectrometer

A free, open-source DIY spectrometer application that runs entirely in your browser. Point a webcam at a diffraction grating and get a live, calibrated emission spectrum — no installation required.

What you can do

  • Analyze light sources: measure emission lines of fluorescent lamps, LEDs and lasers in real time.
  • Calibrate the wavelength axis with multi-point trim against the 436/546 nm mercury lines.
  • Detect and label spectral peaks and dips automatically.
  • Export spectra as CSV/TXT files compatible with Theremino Spectrometer for Windows.
  • Use TCD1304/TCD1254 linear CCD sensors over Web Serial for higher resolution.

Requires the latest Google Chrome. Works on Linux, macOS and Windows. UI and features are inspired by Theremino Spectrometer; this is an independent web implementation licensed under GNU GPL v3.
