Web Spectrometer

A free, open-source DIY spectrometer that runs entirely in your browser. Point a webcam at a diffraction grating and get a live, calibrated emission spectrum — no installation required.

  • Analyze light sources: emission lines of fluorescent lamps, LEDs and lasers in real time
  • Multi-point wavelength calibration against the 436/546 nm mercury lines
  • Automatic peak/dip detection, CSV/TXT export compatible with Theremino Spectrometer
  • TCD1304/TCD1254 linear CCD sensors supported via Web Serial

🔒 Your privacy: all image processing happens locally in your browser — no data is ever uploaded.

Runs in Chromium-based browsers (Chrome, Edge, Opera…); Firefox and Safari are not supported (no Web Serial) · Works on Linux, macOS and Windows · GNU GPL v3. An AI-assisted implementation by Jayden, created with reference to Theremino Spectrometer.
